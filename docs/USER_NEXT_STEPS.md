# Que tiene que hacer Rodrigo

## Ahora

1. Probar la app local completa.
2. Confirmar si el nombre final sera Zentra.
3. Confirmar bundle id/package:
   - iOS: `com.zentra.app`
   - Android: `com.zentra.app`
4. Crear una API key de OpenAI para produccion.
5. Elegir hosting del backend.

## Para publicar web

1. Darme la URL publica del backend cuando exista.
2. Crear cuenta en Vercel, Netlify o Cloudflare Pages.
3. Subir el export web o conectar el repo.
4. Comprar/conectar dominio si queres.

## Para App Store

1. Crear Apple Developer Program.
2. Definir nombre legal/desarrollador.
3. Preparar screenshots.
4. Publicar politica de privacidad.
5. Revisar aviso medico y terminos.

## Para Play Store

1. Crear Google Play Console.
2. Completar ficha de tienda.
3. Subir screenshots.
4. Completar Data Safety.
5. Publicar politica de privacidad.

## Para Supabase

1. Crear proyecto.
2. Ejecutar `docs/SUPABASE_SCHEMA.sql`.
3. Copiar URL y anon key.
4. Pasarme esos datos o ponerlos en `.env`.

## Para pagos PRO

Decision recomendada:

- RevenueCat para iOS/Android.
- Stripe para web.

Necesitas:

- Cuenta Apple Developer activa.
- Cuenta Google Play activa.
- Cuenta Stripe si vas a cobrar en web.
- Definir precio mensual/anual.
